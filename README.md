# Starbucks Clone

A front-end clone of the Starbucks Coffee Company website, built as a demonstration of modern web design and responsive UI techniques. This project replicates key visual components and interactive features of the official Starbucks site, including a navigation bar, hero section, product showcase, rewards promotion, and mobile-friendly layout.

## 📸 Preview

> *Hero section with summer promotion, featured product cards, story and secret menu sections, rewards banner, and a full footer.*

## ✨ Features

- **Fully Responsive Design** – Adapts seamlessly to desktop, tablet, and mobile devices.
- **Mobile Hamburger Menu** – Toggle navigation links and user actions on smaller screens.
- **Smooth Scrolling** – Anchor links scroll smoothly to target sections.
- **Interactive Product Cards** – Hover effects with scale transitions and shadow enhancement.
- **Hero Section** – Split layout with text call‑to‑action and background image.
- **Featured Products Grid** – Displays three coffee / beverage items with images and descriptions.
- **Story & Secret Menu Sections** – Highlight limited‑edition collaborations and app‑exclusive drinks.
- **Rewards Banner** – Encourages users to join Starbucks Rewards.
- **Legal Disclaimer** – Includes sample promotional terms and conditions.
- **Footer** – Multi‑column links and copyright notice.
- **Accessibility** – Focus outlines on interactive elements for keyboard navigation.

## 🛠️ Technologies Used

- **HTML5** – Semantic markup.
- **CSS3** – Custom properties (variables), Flexbox, Grid, animations, and media queries.
- **JavaScript (Vanilla)** – DOM manipulation for mobile menu toggle and smooth scrolling.
- **Google Fonts** – *SoDoSans* (Starbucks-like typeface).
- **SVG Icons** – Inline store locator icon.

## 🚀 Getting Started

Follow these steps to run the project locally:

1. **Clone the repository**

   git clone https://github.com/your-username/starbucks-clone.git
   
2. **Navigate into the project folder**
 
   cd starbucks-clone

3. **Open `index.html` in your browser**
   - Double-click the file, or
   - Use a local development server (e.g., [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code)

No build steps or dependencies are required – it’s pure HTML/CSS/JS.

## 📱 Responsive Breakpoints

- **≤ 900px** – Hero switches to column layout, mobile menu button appears.
- **≤ 768px** – Story and Secret Menu sections become single column.
- **≤ 600px** – Product grid becomes single column, typography scales down.
